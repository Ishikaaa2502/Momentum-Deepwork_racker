Momentum is a mobile-first productivity web application designed to help users build deep focus habits using a Pomodoro-based approach.

Live Demo : https://delicate-axolotl-f221ec.netlify.app/

---

## Problem Statement

Many productivity tools are overloaded with features and visual clutter. This increases cognitive load instead of improving focus. Momentum provides a clean and distraction-free interface to encourage deep work.

---

## Key Features

- 25-minute focus timer
- Pause and reset functionality
- Automatic session tracking
- Productivity score calculation
- LocalStorage data persistence
- Dark mode toggle
- Fully responsive layout

---

## Tech Stack

- HTML5
- CSS3 (Flexbox + Media Queries)
- Vanilla JavaScript
- LocalStorage API

---

## Design Approach

The project follows a mobile-first strategy. The layout was initially designed for small screens and later optimized for tablets and desktops using media queries.

The interface focuses on visual clarity, consistent spacing, readable typography, and minimal color usage to support focused productivity.
